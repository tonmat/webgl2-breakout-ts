# webgl2-breakout-ts

A simple breakout game using WebGL2

## Install

```
yarn
```

## Run

```
yarn start
```

Open a browser at http://localhost:3001

## Controls

<table>
    <tr>
        <td><kbd>ALT</kbd> + <kbd>SCROLL LOCK</kbd></td>
        <td>start the game</td>
    </tr>        
    <tr>
        <td><kbd>MOUSE</kbd></td>
        <td>move the player</td>
    </tr>            
</table>
